  2. 已安裝 Devonthink
  3. 沒有了…

操作過程#

  1. 打開Devonthink，File -> Import -> Note from Evernote

  2. 選擇想要匯入的資料夾

  3. 結束，是不是超級簡單啊?! 連TAG都幫您一同移轉過來了，完全無痛轉移。
